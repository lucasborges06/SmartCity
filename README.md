# SmartCity
SmartCity Vision** é um protótipo de site futurista e interativo que apresenta soluções para cidades inteligentes. O projeto tem como objetivo demonstrar, de forma visual e inovadora, como tecnologias avançadas podem ser aplicadas para melhorar a sustentabilidade, segurança, gestão urbana e qualidade de vida nos centros urbanos.

O site combina **design moderno**, **animações futuristas**, **gráficos interativos** e **partículas dinâmicas**, oferecendo uma experiência imersiva para o usuário. É apenas um protótipo visual, sem backend implementado, mas com grande potencial para expansão.

---

## Funcionalidades

- Vídeo de fundo em tela cheia cobrindo toda a página.
- Navbar animada e responsiva com efeitos neon.
- Seções informativas sobre a empresa, equipe e serviços.
- Cards interativos com hover futurista e levitação.
- Gráficos interativos usando **Chart.js** para simular painéis de cidade inteligente.
- Partículas animadas com **Particles.js**.
- Scroll suave e efeitos parallax leves.
- Layout totalmente responsivo e moderno.

---

## Tecnologias Utilizadas

- **HTML5** e **CSS3**
- **JavaScript (ES6)**
- **Bootstrap 5** – framework CSS para responsividade
- **Chart.js** – gráficos interativos
- **Particles.js** – animação de partículas
- **AOS (Animate On Scroll)** – animações ao rolar a página
- Vídeo de fundo em **HTML5 `<video>`**
